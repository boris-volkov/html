Here is an example of a page layout using css and html.

The way the code is written now assumes that you have 
the following three files in your directory:
	page.html  		<- containing the html text below
	style.css  		<- containing the css text below(er)
	picture.jpg		<- being some picture

These names are not required, but whatever name you choose
has to match what you write in the html.

This layout is just here for you to practice getting a page working. 
Choose your own topic, play around with the numbers, change the colors,
change the borders/padding/margins to how you like it. Make it your own!
There is still much to be done to make this page look great...
./soccer_page/page.html
<!DOCTYPE html>

<!-- the text between these arrow things is a comment  -->
<!-- it doesn't show up on the page in any way, but is -->
<!-- just a way for you to leave notes for anyone who  -->
<!-- may be reading your code, including yourself in   -->
<!-- the future when you forget how everything works!  -->

<html>
	<head>
		<title>soccer</title>
		<link rel="stylesheet" href="style.css"></link>
		<!-- the link tag points to the css file that controls the style 	-->
		<!-- it can be in the same directory together with your html file 	-->
		<!-- this is similar to having everything in style tags, 			-->
		<!-- but better, since multiple pages can use the same css style 	-->
	</head>

	<body>
		<div class="outer">
		<!-- I give the divs class names so that I can target them   -->
		<!-- separately and give them different styles and behaviors -->
			<img src="picture.jpg"></img>
			<div class="inner">
				soccer is an excellent sport...<br>
				here are the six pillars of success in soccer:
				<ol> <!-- starting an ordered list -->
					<li> strong kicking 		</li>
					<li> swift running 			</li>
					<li> accurate passing 		</li>
					<li> controlled dribbling 	</li>
					<li> expansive field control</li>
					<li> staunch guarding   	</li>
				</ol>
			</div>
		</div>
	</body>
</html>
./soccer_page/style.css
/* css comments come between these slash/star things 				*/
/* leave yourself notes about anything you think you might forget!  */
/* look up css properties at https://www.w3schools.com/csSref/      */

body {                     	/* properties of the entire body */
	background-color: rgb(40,95,70);
}

img {					    /* properties of any img on the page */
	width : 600px;
	height : 600px;
	margin : 20px;			
	object-fit : cover; 	/* to crop into image */
	border-radius : 50%; 	/* to round the edges */
	float : left;			/* so text can go around it */
	border-style : solid;
}

div { 						/* properties that all divs will have */
	color : #fff;
	border-style : solid;
	margin : 20px;
	font : Courier;
	font-weight : 300;
}

div.outer { 				/* selects only divs with class="outer" */
	margin : auto;
	background-color : rgb(20,30,60);
	overflow : auto; 		/* needed when all internal elements are float */
	display : flex;  		/* to vertically center internal divs */
	align-items : center; 	/* vertical centering */
}

div.inner {					/* selects only divs with class'"inner" */
	padding : 20px;
	background-color : rgb(40,70,200);
	font-size : 24px;
	float : left;
}
