Here is an example of a page layout using css and html. 
The html is in the green box, the css is in the teal box.

The way the code is written now assumes that you have 
the following three files in your directory:
	page.html  		<- containing the html text
	style.css  		<- containing the css text
	picture.jpg		<- being some picture

These names are not required, but whatever name you choose
has to match what you write in the html.

This layout is just here for you to practice getting a 
page working. Play around with the numbers, change the colors,
change the borders/padding/margins to how you like it.
Make it your own!

<!DOCTYPE html>

<html>
	<head>
		<title>soccer</title>
		<!-- the next line points to the css file that controls the style -->
		<!-- it can be in the same directory together with your html -->
		<link rel="stylesheet" href="style.css"></link>
	</head>

	<body>
		<div class="outer">
			<img src="picture.jpg"></img>
			<div class="inner">
				soccer is about kicking.
			</div>
		</div>

	</body>

</html>

body {
	background-color: rgb(40,95,70);
}

img {
	width : 600px;
	height : 600px;
	margin: 20px;
	padding : 20px;
	object-fit: cover; 		/* to crop into image */
	border-radius: 50%; 
	float: left;			/* so text can go around it */
	border-style:solid;
}

div.outer { 				/* selects all divs with class="outer" */
	margin: auto;
	background-color: rgb(20,30,60);
	color: white;
	border-style:solid;
	margin: 20px;
	overflow: auto; 		/* needed when all internal elements are float */
	display: flex;  		/* to vertically center internal divs */
	align-items: center; 	/* vertical centering */
}

div.inner {
	margin: 20px;
	padding: 20px;
	background-color: rgb(40,70,200);
	border-style:solid;
	color:    white;
	font-size: 72px;
	float: left;
}
